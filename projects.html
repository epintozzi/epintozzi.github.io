US Title Series

My first freelance project post-Turing, this was an especially exciting one. I project managed and built this site start to finish as a solo endeavor. The US Title Series boat racing club had some interesting business requirements as part of shifting from a paper-only organization to a digital one. The most important features include News postings up front to keep members up to date, and well as allowing users to register and pay for club membership and races online. While the end goal is to move everything online, the registration process allows for payments to be deferred and paper applications to still exist during their transition which was important to this group.

  • Ruby/Rails
  • Paypal Express payments
  • Photo storage with S3
  • Heroku and Postgres
  • TDD
  • Design/UX and implementation with Bootstrap 3
  • Project Management with waffle.io

Tuesdates

Tuesdates uses the Yelp API to make restaurant recommendations based on a user search and send email alerts via SendGrid. This was a solo project that I managed from the wireframe/mockup phase through deployment.

  • Ruby/Rails
  • Yelp API
  • Google OAuth
  • HTML/CSS/Bootstrap 3
  • TDD
  • Design/UX
  • Project Management with waffle.io

Github Clone

This project focuses on consuming and working with data from public APIs. I have recreated a version of the Github user profile page where users can see their recent activity, repositories, starred repositories, followers, and those they follow.

  • Ruby/Rails
  • OAuth Login
  • Github API
  • HTML/CSS/Bootstrap 3
  • TDD with Rspec, Capybara, and Factorygirl
