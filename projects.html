Tuesdates

Tuesdates uses the Yelp API to make restaurant recommendations based on a user search and send email alerts via SendGrid. This was a solo project that I managed from the wireframe/mockup phase through deployment.

  • Ruby/Rail
  • Yelp API
  • Google OAuth
  • HTML/CSS/Bootstrap 3
  • TDD
  • Design/UX
  • Project Management with waffle.io

Github Clone

This project focuses on consuming and working with data from public APIs. I have recreated a version of the Github user profile page where users can see their recent activity, repositories, starred repositories, followers, and those they follow.

  • Ruby/Rail
  • OAuth Login
  • Github API
  • HTML/CSS/Bootstrap 3
  • TDD with Rspec, Capybara, and Factorygirl
